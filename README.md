# PlatformGame

Instructions: Use the left/right arrows to move in either direction. Space to jump. Avoid all monsters as touching them will restart the game. Get to the grass platform to win. Try to collect all 8 coins!

To work on the game in Unity:
1) Download the asset package: https://github.com/AdamBadagliacco/PlatformGame/blob/master/PlatformGameAssetPackage.unitypackage
2) Create a new Unity project and delete the sample scene "SampleScene.unity" 
3) In Unity click: Assets > Import Package > Custom Package 
4) Import all assets
5) Open the newly imported sample scene and the project will open in unity to either be edited or played however you desire. 


To play the game:
1) Download the release found here: https://github.com/AdamBadagliacco/PlatformGame/releases/tag/v1.0
2) Unzip and run "PlatformGame.exe"
